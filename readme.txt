Pull feed items as posts into your WordPress site. Automatically create/update posts when feed changes.

== Description ==

Easily setup feeds to be syndicated into WordPress. You can map feed attributes to post fields or post meta. Syndicated
content can be mapped to any post type you want. The plugin comes with a convenient settings page so you can configure
things like when feeds are syndicated. Plugin functionality is very extensible.

== Installation ==

1. Upload plugin folder
1. Activate
1. In the WordPress backend, create a new "Source Feed" that maps to a feed you need syndicated.
